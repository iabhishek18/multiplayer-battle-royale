# Multiplayer Battle Royale Game

100-player battle royale with Unity, Photon PUN networking, shrinking zones, and loot system.

## Features
- 🎮 100 concurrent players via Photon PUN
- 🗺️ Shrinking safe zone (6 phases)
- 🔫 Multiple weapons with ballistic physics
- 🎒 Loot spawning and inventory system
- ❤️ Health + Shield system
- 📡 Network-synced player states
- 🏆 Kill feed and leaderboard

## Tech Stack
- **Engine**: Unity 2022 LTS
- **Networking**: Photon PUN 2
- **Language**: C#
- **Server**: Photon Cloud

## Getting Started
1. Open in Unity 2022.3+
2. Import Photon PUN 2 from Asset Store
3. Set your Photon App ID in PhotonServerSettings
4. Build and run
